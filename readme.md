# Fake News Website Project
## Environment setup
nvm version: 0.35.2
npm version: 7.18.1
node version: 16.4.0

to set up the env:
```shell
git clone https://github.com/dinghr537/fake-news-website.git

cd fake-news-website

# make sure you're using the correct version of nvm/npm/node
npm install
```

## Production build
```shell
npm run build
```

## Production dev
```shell
npm run dev
```
